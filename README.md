# DDEX

DDEX es una libreria para graficos 2D sobre windows basada en plugins para las distintas apis graficas.

Característica

- Carga dinamica de imagenes
- Efectos al dibujar imagenes
- Efectos de presentacion
- Efectos de iluminacion
- Captura de imagenes(screenshot) en segundo plano
- Manejo de transiciones de colores 
  
### Effectos de dibujado
- Alphablending
- Invertir colores
- Escala de grises y color
-Invertir imagne
- Relieve
- Sacudida(Swinging)
- Onda
- Rotacion
 

### Effectos de presentacion
- Invertir colores
- Escala de grises y color
-Invertir imagne
- Relieve
- Sacudida(Swinging)
- Onda
- Rotacion
- Temblor
- Vibracion
- Borracho(beta)
- Flash(beta)


### Desarrollador
Es un proyecto en C++/C# con Visual Studio 2013.

La libreria principal no tiene niguna dependencia.

###### DX9
Solo usa el [SDK de DirectX](https://www.microsoft.com/en-us/download/details.aspx?id=6812)
